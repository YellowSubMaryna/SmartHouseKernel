Описание содержимого папки:

SmartHouseKernel.exe               - исполняемый файл проекта, скомпилирован в VS2022
myroom.bin                         - файл в котором между запусками программы хранится набор датчиков в бинарном виде
..\Music                              - каталог с музыкой, проигрываются mp3 файлы из этого и всех вложенных каталогов

SDL2.dll                - библиотека для создания игр, используется для проигрывания mp3 файлов
SDL2_mixer.dll

libegl.dll              - кодеки для проигрывания музыкальных файлов других форматов библиотекой SDL
libFLAC-8.dll             (для уменьшения объёма остальные кодеки не копируются,
libmpg123-0.dll            при необходимости можно скачать их вместе с исходниками библиотеки SDL)
libogg-0.dll
libvorbis-0.dll
libvorbisfile-3.dll
